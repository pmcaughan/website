Things I've done

Machine Learning Diabetes Predictor

This uses decision tree machine learning to predict whether or not someone has diabetes given a handful of diagnostic measurements. This was trained and tested using this public data set of Pima Indians from Kaggle and had about 63% accuracy. I plan to try to re-implement this using a random forest, rather than a single decision tree in the future, to see if I can do better. This was my first real machine learning project, and the experience of learning some techniques in machine learning outside of school was a unique experience that I felt was especially valuable.

Neural Network Digit Recognition

A basic handwritten digit recognition program written in python, made using the Keras API and trained using the MNIST data set. This was a way to learn about the basics of implementing machine learning and a way to learn more about the internal workings of neural networks.

Chillennium Game Jam

More an experiment in how quickly I can figure something out by myself than anything, I participated in the 2018 Chillennium Game Jam, the largest student-run game jam. I had 24 hours to put together a game, and the theme of that year's jam was "Transformation". This was one of the most fun projects I've done, as I had zero experience making games, using the Unity engine, or using the C# language, so I had to learn as quickly as possible to make something playable.
